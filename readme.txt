Removes unnecessary external links from both admin and front end admin bars when using Yoast WordPress SEO plugin.

== Description ==

Yoast produces some excellent plugins, Kudos to the man, but my clients don't need or want to see his SEO links in their admin bar.
Remove them with this plugin.

== Installation ==
 
Upload in normal way, to your `/wp-content/plugins/` folder and activate.  

1. Upload `Clean-My-Bars` folder to your `/wp-content/plugins/` directory  
2. Just activate as normal, no settings required.
